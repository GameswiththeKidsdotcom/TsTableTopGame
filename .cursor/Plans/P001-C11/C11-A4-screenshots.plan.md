## Outcome

Screenshots uploaded for each required device size in App Store Connect.

## Required sizes

- iPhone 6.7" (e.g. 14 Pro Max): 1290 x 2796 px
- iPhone 6.5" (e.g. 11 Pro Max): 1242 x 2688 px
- iPad Pro 12.9" (target 2): 2048 x 2732 px

## Steps

1. Run app in Xcode Simulator for each device size.
2. Capture 3–5 screens: menu, gameplay (both boards), game over overlay, settings.
3. Upload in App Store Connect > App Store > [Version] > Screenshots.

## Validation

- Screenshots accepted for each size with no errors.
- No placeholder or generic images.

## Rollback
